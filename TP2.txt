1. Ecrire un script "script1.sh" qui propose à l'utilisateur de saisir un chiffre.
Ce chiffre sera statiquement défini dans le script (exemple: 85).
En fonction du nombre saisi par l'utilisateur, le script indiquera "c'est plus" ou "c'est moins" jusqu'à ce que le chiffre correct soit trouvé. Le script affichera alors "Bravo, tu as trouvé le chiffre secret" et s'arrêtera.

2. Ecrire un script "script2.sh" prenant une url en argument de commande.
Le script fera une requête http sur cette url et indiquera dans le terminal 
ainsi que dans un fichier de log la date, l'url et statut réponse.
Exemples:
04/08/2022 http://example.com 200
04/08/2022 http://example.com/toto 404
